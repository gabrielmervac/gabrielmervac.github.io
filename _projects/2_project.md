Transcription factor dynamics carry enough information to **predict future cellular responses**. We built LSTM neural network models and mechanistic frameworks that map NF-κB dynamics — measured in real time via live-cell imaging — to downstream transcriptional outcomes and subsequent stimulation responses.

A key finding is the role of **IFN-β as a paracrine memory amplifier**: primary responders release IFN-β, which acts as a quorum-like signal that primes neighboring cells and shapes population-level immune memory. This creates a cell-intrinsic and cell-extrinsic memory circuit that can be read out and predicted from single-cell transcription factor traces.
